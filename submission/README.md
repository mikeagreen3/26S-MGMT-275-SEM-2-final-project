# Submission — Schlage AI Access Concierge

The four graded deliverables for the MGMT 275 final project. Suggested reading order:

1. [`prfaq.md`](./prfaq.md) — Press release and FAQ for the Schlage AI Access Concierge launch. Start here for the narrative and the customer-facing rationale.
2. [`source-of-truth.md`](./source-of-truth.md) — Refined spec for the alert-copy agent: system prompt, input schema, validator rules, fallback behavior, and the channel-output contract that the React prototype enforces.
3. [`ux-eval-appendix.md`](./ux-eval-appendix.md) — Interview synthesis from 10 AI-simulated user interviews + eval-set design. Cross-cutting findings, recommendations, and pricing analysis.
4. [`interviews/`](./interviews/) — Methodology, source corpus, 10 source-pinned personas, 7-section interview guide, and per-persona syntheses (`host-a` through `host-j`). Raw transcripts live in `interviews/raw/` (gitignored).

Eval harness, eval set, and result CSVs are in the sibling [`../eval/`](../eval/) folder. Run instructions and the working prototype are in the root [`../README.md`](../README.md).
